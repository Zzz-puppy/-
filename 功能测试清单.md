## 一、用户模块

### 1.1 一键登录
- [ ] 未登录状态下进入"我的"页面，显示"一键登录"按钮
- [ ] 点击"一键登录"，无需输入昵称，自动完成登录
- [ ] 登录成功后显示用户信息（昵称、信用分等）
- [ ] 重启后端后再次一键登录，仍是同一账号但**认证状态被重置为未认证**（昵称不变）
- [ ] 登录成功后可以正常使用需要认证的功能

### 1.2 清除登录
- [ ] 点击"清除登录信息"，确认后退出登录状态
- [ ] 清除后页面恢复为未登录状态
- [ ] 清除后需要重新登录才能使用受限功能

### 1.3 编辑资料
- [ ] 点击用户头像区域，弹出编辑昵称弹窗
- [ ] 输入新昵称并保存，页面实时更新
- [ ] 空昵称应提示"昵称不能为空"

### 1.4 校园认证
- [ ] 未认证用户显示"校园认证"入口
- [ ] 点击后依次输入学号和校园邮箱
- [ ] 提交后显示"认证审核中"
- [ ] 已认证用户显示学号和"已认证"标签
- [ ] 管理员无需认证，不显示认证入口
- [ ] 认证通过/拒绝后，在"消息－通知"中收到通知

- [ ] 分类切换（数码/图书/生活/运动/服装/其他）能筛选商品

- [ ] 详情页显示：商品图片、标题、价格、描述、卖家信息、标签

### 2.3 搜索功能
- [ ] 点击搜索框进入搜索页

- [ ] 商品正确显示状态：**在售**（绿色）、**已售出**（灰色）、**已下架**（橙色）
- [ ] 在售商品可以点击"售出"（自己的商品才能操作）
