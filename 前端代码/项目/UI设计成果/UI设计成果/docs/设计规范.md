# 校易助设计规范

## 色彩

- 主色：`#15B889`，用于主按钮、选中状态和核心行动。
- 深主色：`#078867`，用于标题强调和 hover 状态。
- 辅助蓝：`#3B82F6`，用于可信、科技感和渐变辅助。
- 警示红：`#EF5B5B`，用于举报和危险操作。
- 暖提示色：`#FFB547`，用于待审核和提醒状态。
- 背景色：`#EEF6F3`，用于页面整体背景。
- 卡片色：`#FFFFFF`，用于商品、表单和信息模块。
- 边框色：`#DCE9E5`，用于卡片、输入框和分割线。

## 字体

- 中文字体优先使用 Microsoft YaHei 或 PingFang SC。
- 页面标题：24px 至 25px，加粗。
- 模块标题：20px，加粗。
- 商品标题：15px 至 16px，加粗。
- 正文说明：13px 至 14px。
- 标签文字：12px，加粗。

## 布局

- 移动端优先，原型宽度模拟微信小程序手机屏幕。
- 页面左右边距约 18px。
- 卡片圆角统一为 8px。
- 卡片之间间距 10px 至 14px。
- 底部固定五栏导航，保证核心功能随时可达。

## 组件

### 按钮

- 主按钮：绿色实底白字，用于发布、联系卖家、提交认证。
- 次按钮：浅绿色底深绿色字，用于搜索、筛选、辅助操作。
- 危险按钮：红色实底白字，用于匿名举报。
- 文本按钮：透明背景，用于返回、查看更多。

### 商品卡片

商品卡片必须包含：

- 商品图片或图片占位
- 商品标题
- 价格
- 分类或服务标签
- 成色
- 交易地点
- 卖家昵称
- 信用分
- 收藏或联系入口

### 标签

标签用于快速展示交易信任信息，例如：

- 本校认证
- 可验货
- 当天交易
- 线下自提
- 可置换

### 表单

- 输入框高度不低于 44px，便于移动端点击。
- 必填信息包括标题、分类、价格、地点和描述。
- 提交后必须有明确反馈。

### 举报与审核状态

- 举报入口使用红色危险按钮，强调风险操作。
- 举报流程必须突出匿名、证据、系统初筛和管理员复核。
- 管理员状态使用徽章展示，便于快速识别待处理数量。

## 可用性原则

- 买家从发现商品到联系卖家不超过三步。
- 卖家从进入发布页到提交商品不超过三步。
- 重要状态必须可见，包括实名认证、信用分、交易地点和举报反馈。
- 避免长段文字堆叠，优先使用卡片、标签和状态徽章。
