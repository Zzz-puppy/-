# 校易助 UI 设计说明

## 设计目标

校易助是一款面向大学生的校园二手物品循环交易微信小程序。UI 设计重点是降低发布和购买门槛，让学生在本校认证环境中快速完成"找货、联系、线下成交"。

核心目标：

- 强化校园身份可信感，突出"本校认证、信用分、当面验货"。
- 缩短交易路径，让发布或购买尽量在三步内完成。
- 建立清晰的商品分区，解决微信群和表白墙信息碎片化问题。
- 通过匿名举报与管理员审核台，体现平台安全治理能力。

## 用户角色

- 买家：快速搜索、筛选、查看商品真实性，联系卖家线下验货。
- 卖家：用模板快速发布闲置，展示信用和交易地点，提高成交效率。
- 管理员：审核身份、处理举报、维护分类标签和违规商品。

## 视觉风格

整体采用移动端优先的小程序界面风格。主色为清爽绿色，辅助蓝色用于科技感和可信感，局部暖色用于提示和审核状态。

界面避免复杂装饰，重点使用卡片、标签、信用徽章、状态提示和底部导航，让用户能快速识别当前任务。

## 核心页面说明

- 首页：展示平台价值、三步交易流程、高效分区和推荐闲置。
- 分类检索：提供关键词搜索、分类筛选、排序和商品卡片。
- 商品详情：突出价格、成色、交易地点、卖家信用和联系按钮。
- 发布商品：使用表单模板完成标题、分类、价格、成色、地点和描述填写。
- 求购区：支持发布需求和智能匹配历史卖家。
- 消息页：模拟联系卖家后的交易沟通提醒。
- 个人中心：展示实名认证、信用分、发布、成交和收藏数据。
- 实名认证：体现学号和校园邮箱双重认证。
- 举报页：支持匿名举报、证据说明和截图上传。
- 管理员审核台：展示身份审核、举报工单和分类维护。

## 个人 UI 设计成果说明

本次 UI 设计成果包括移动端高保真原型、页面交互、用户流程图、交互说明和视觉设计规范。设计重点围绕校园二手交易的可信、安全、高效展开，保证开发同学可以根据原型理解页面结构、交互逻辑和核心状态。
