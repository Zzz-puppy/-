# 校易助交互说明文档

## 全局导航

- 底部导航包含：首页、分类、发布、求购、我的。
- 点击导航后切换到对应页面，并同步顶部标题。
- 顶部消息按钮点击后显示交易提醒反馈。

## 首页

- "发布闲置"按钮跳转到发布页面。
- 搜索框输入关键词后点击搜索，跳转到分类检索页并展示筛选结果。
- 点击分类卡片进入分类检索页，并自动选中对应分类。
- 点击推荐商品图片进入商品详情。

## 分类检索

- 支持关键词搜索、分类标签筛选和排序。
- 排序包含最新发布、价格从低到高、信用优先。
- 商品卡片展示标题、价格、标签、成色、地点、卖家和信用分。
- 收藏按钮点击后切换收藏状态，并显示反馈提示。
- "看详情"进入详情页，"联系卖家"进入消息页。

## 商品详情

- 展示商品主图占位、标题、价格、分类、交易地点、标签、描述和卖家信用。
- "联系卖家"按钮用于模拟安全联系，只在买家主动联系后进入消息页。
- "举报"按钮跳转到匿名举报页面。

## 发布商品

表单字段：

- 商品标题
- 分类
- 价格
- 成色
- 交易地点
- 商品描述

提交后显示"发布成功"反馈，说明商品会展示本校认证、信用分和自提地点。

## 求购区

- 用户可以填写求购需求并发布。
- 发布后需求进入列表，状态显示"正在智能匹配"。
- "智能匹配"按钮用于模拟系统匹配历史卖家和相似商品。

## 个人中心与实名认证

- 个人中心展示头像、昵称、年级专业、认证状态、信用分、发布数、成交数、收藏数。
- 点击"实名认证"进入认证页。
- 认证页需要填写学号、校园邮箱和验证码。
- 提交后显示认证申请已提交。

## 精细化举报

- 举报类型包括虚假商品、违规品类、恶意引流和交易纠纷。
- 用户需要填写证据说明，可上传证据截图。
- 提交后显示匿名举报成功，并进入系统初筛和管理员审核流程。

## 管理员审核台

- 展示身份审核、举报工单和下架商品统计。
- 管理员可以对审核项执行通过、下架、更新等模拟操作。

## 异常与提示

- 求购需求为空时提示"请先填写求购需求"。
- 搜索无结果时展示空状态，引导用户发布求购。
- 所有关键操作使用 Toast 反馈，避免用户不确定操作是否成功。
