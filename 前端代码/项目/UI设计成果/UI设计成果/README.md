# 校易助 UI 设计成果包

本成果包用于软件工程课程设计展示，内容围绕"校易助：校园二手物品循环交易平台"的微信小程序 UI 原型与设计说明展开。

## 文件说明

- `prototype/index.html`：静态高保真原型入口。
- `prototype/styles.css`：视觉样式、移动端布局和组件规范。
- `prototype/app.js`：搜索、筛选、详情、收藏、表单反馈等交互。
- `docs/UI设计说明.md`：UI 设计目标、用户角色、页面设计说明。
- `docs/用户流程图.md`：买家、卖家、管理员 Mermaid 流程图。
- `docs/交互说明文档.md`：页面操作、按钮反馈和异常状态说明。
- `docs/设计规范.md`：颜色、字体、按钮、卡片、表单等 UI 规范。

## 展示建议

汇报时可以重点展示首页、分类检索、商品详情、发布商品、求购区、举报页面和管理员审核台，突出"本校认证、线下自提、三步交易、精细化举报"的设计亮点。
